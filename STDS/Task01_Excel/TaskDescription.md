# Excel Dashboard Project: Data Science Job Market Analysis

## 1. Project Overview

This project involves analyzing a dataset of **Data Science job postings for the year 2025**. The dataset provides insights into the current job market, covering various aspects such as job titles, salaries, required skills, company details, and location types (Remote, Hybrid, On-site).

## 2. Dataset Description

The dataset consists of **944 records** and includes the following key columns:

- **Job Details:** Job Title, Seniority Level, Status (Hybrid/On-site/Remote).
  
- **Company Info:** Company Name, Industry, Size, Revenue, Headquarter.
  
- **Requirements:** Key technical skills (Python, SQL, AWS, etc.).
  
- **Financials:** Average Salary estimates.

## 3. Objectives

The main objective of this task is to utilize **Microsoft Excel** to perform end-to-end data analysis, from data cleaning to visualization.

The final output is an interactive **Dashboard** designed to assist job seekers and recruiters in understanding the 2025 Data Science market landscape. Specifically, the dashboard aims to:

- **Analyze Salary Drivers:** Investigate how Average Salary varies across different Seniority Levels, Industries, and ownership.
  
- **Work Mode Trends:** Visualize the distribution of work types (Remote vs. Hybrid vs. On-site) to understand current workplace flexibility.
  
- **Company Insights:** Visualize Frequency of Ownership of companies and Revenue Status.
